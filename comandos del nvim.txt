Espacio  + c es para borrar los resultados de una busqueda.

Para hacer la busqueda se hace el / y luego lo que se busca.

Las teclas de movimiento son hjkl pero para salir del modo insertar, se apreta j y k a la vez.

Espacio + + para aumentar números y Espacio + - para disminuir.

Espacio + sv para separar verticalmente la pantalla
Espacio + sh para separar horizontalmente la pantalla
Espacio + si para hacer las ventanas iguales en alto y ancho
Espacio + sc para cerrar las ventanas separadas
Espacio + sm para maximizar una ventana 


Espacio + to para abrir una nueva tab
Espacio + tc para cerrar la tab actual
Espacio + ne para ir a la siguiente tab
Espacio + an para ir a la anterior tab


Cuando la pantalla esté separada puedo usar control hjkl para moverme entre las pantallas separadas

ysw algun caracter para encerrar una palabra entre ese carcater
ds para quitarlo seguido del caracter
cs para cambiar el caracter que encierra seguido del caracter a cambiar y seguido del nuevo caracter

yw para copiar texto 
w para moverse a la siguiente palabra(aplica siempre)
grw para cambiarlo sobre algo que ya esta


gcc para comentar una fila
gc(fila)j para comentar hasta la fila
para deshacer es lo mismo

Espacio + e para abrir el explorador de archivos
shift + g para ir al final del archivo 
shift + f para ir al inicio del archivo

a para añadir un nuevo archivo en la carpeta actual

Control + O para ir al anterior paso
Control + I para ir al siguiente paso

Espacio + ff para buscar archivos
Espacio + fc o fs para buscar texto en los archivos

Espacio + tt para dividir la pantalla verticalmente y abrir una terminal

Para hacer scroll:
	Control + e para bajar una línea
	Control + y para subir una línea
	Control + f para bajar una página
	Control + b para subir una página
	Control + d para bajar media página
	Control + u para subir media página



* sobre alguna palabra para buscar esa palabra
n para moverse al siguiente resultado
N para moverse al anterior resultado

para reemplazar es:
	:s/antiguoTexto/nuevoTexto/g		Para el resultado en el que está el cursor
	:%s/antiguoTexto/nuevoTexto/g		Para todos los resultados del antiguoTexto
	:%s//nuevoTexto/g					Para todos los resultados de la última búsqueda

Shift + k para buscar en las páginas de manuales la función que está sobre el cursor
